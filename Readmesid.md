# Dlightful Saga 📚
A full-stack digital bookstore platform designed to host, showcase, and sell fictional books.

This project is divided into:
- 🎨 Frontend (React + TailwindCSS)
- 🔧 Backend (Node.js + Express + MongoDB)
- 🔐 Authentication (JWT & Google OAuth)
- 💰 Payments (Stripe, Paytm, GPay — Phase 2)

---

## 📁 Project Structure
Dlightful_Saga/
├── Backend/ # Express server, MongoDB logic, routes, JWT, etc.
├── Frontend/ # React UI, TailwindCSS styling, routing
├── README.md # Project overview

## 🚀 Getting Started

### Prerequisites
- Node.js
- MongoDB (Community version installed locally or MongoDB Atlas)
- Postman (optional for testing backend APIs)
