# Dlightful Saga – Frontend 🎨

This is the frontend client for the Dlightful Saga bookstore, built with **React + TypeScript** and styled using **TailwindCSS**.

---

## ⚙️ Tech Stack

- React + TypeScript
- TailwindCSS
- React Router DOM
- Vite
- Google OAuth (coming in Phase 2)

---

## 🧱 UI Features

- Login / Sign up forms with validation
- Password reset flow with feedback card
- Continue with Google button (UI integrated)
- Wishlist toggle
- Dynamic book UI placeholders
