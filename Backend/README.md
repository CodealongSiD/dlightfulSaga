# Dlightful Saga – Backend 🔧

This is the backend API for the Dlightful Saga bookstore. Built with **Node.js, Express, MongoDB, and JWT**, this API handles user authentication, book data, and payment (in future phase).

---

## ⚙️ Tech Stack

- Node.js
- Express
- MongoDB (via Mongoose)
- JWT for Auth
- Postman (for testing APIs)
- Nodemailer (for forgot password)
- Stripe SDK (later)
